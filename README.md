### 🪐 the mission
I build autonomous systems that bridge the gap between **digital reasoning** and **physical environment manipulation**. My focus is at the intersection of **Deep Reinforcement Learning, Multi-Agent Warehouses, and Multimodal Vision Agents** for robotics and industrial applications.

---

### 🛠️ current research & builds

#### 🏭 [AIDE](https://github.com/Durgaprasad-Developer/AIDE) ── Autonomous Industrial Decision Engine
> *A multi-agent Reinforcement Learning warehouse environment built for industrial policy-driven inference.*
*   Implemented an OpenAI-Gym compliant sandbox environment.
*   Enables agent routing, collision avoidance, and dynamic task allocation in simulated warehouse grid spaces.
*   **Stack:** Python, Gym, PyTorch, Stable-Baselines3, FastAPI.

#### 🧠 [Sortiq](https://github.com/Durgaprasad-Developer/Sortiq) ── Reinforcement Learning From Scratch
> *An RL-based routing monolith built completely from raw math without third-party ML frameworks.*
*   Wrote custom Q-learning and Policy Gradient environments from the ground up to understand reward-shaping mechanics.
*   Designed an intelligent routing loop to dynamically optimize action choices.
*   **Stack:** JavaScript, Math, Node.js.

#### ⚡ [UX-Ray](https://github.com/Durgaprasad-Developer/UX-Ray) & [Git-XRay](https://github.com/Durgaprasad-Developer/Git-XRay) ── Cybernetic Vision Agents
> *Autonomous vision-based simulation loops acting as real-time QA testers.*
*   Integrates Playwright sandboxes and Set-of-Mark visual coordinate overlays.
*   Uses multimodal LLMs (Gemini 2.0 Flash) to read layout coordinates, auto-wait for loading states, and output layout diagnostics.
*   **Stack:** TypeScript, Playwright, Next.js, Gemini Vision APIs.

#### 🎬 [HeyGen-Clone](https://github.com/Durgaprasad-Developer/Heygen-clone) ── Multimodal Lip-Sync Synthesis
> *An automated talking-head AI avatar pipeline.*
*   Connects dynamic voice waveform inputs to visual generation networks.
*   Built out worker pools and video animation pipelines to automate video content creation.
*   **Stack:** Python, Docker, PyTorch, FastAPI, ML Workers.
